# RAG-Workflow-con-LlamaIndex-Ollama-y-HuggingFace-Embeddings
Proyecto de ejemplo que implementa un sistema RAG (Retrieval-Augmented Generation) con LlamaIndex Workflows. El pipeline se divide en ingesta de documentos, recuperación de contexto relevante y síntesis de respuestas. Usa Ollama como LLM y HuggingFaceEmbedding para generar embeddings y consultar documentos locales de forma modular.
